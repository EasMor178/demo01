#### 工作台编辑页面逻辑

# 方向变动

以格子向左增加尺寸为例

监听格子最左侧的一列方块，当最左侧的方块向左增加一格时，循环查询是否有格子的最右侧正好处于移动方块的位置。
如果查询到，则将这个格子的原位置记录，并将这个格子的位置向左移动一格，同时循环查询是否有格子的最右侧正好处于移动方块的位置。
以此类推。


# 面板拖拽

当用户对格子位置发生拖拽行为时，使用函数监听格子的位置
当该格子的位置与其他格子的位置发生重叠时，交换两个格子的坐标。
